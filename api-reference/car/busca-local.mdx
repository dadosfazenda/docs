---
title: "Busca Local por CAR"
api: "GET /car/local/{codigo_car}"
description: "Busca dados do CAR no banco de dados local com cruzamento SIGEF/SNCI"
---

## Descricao

Busca dados do CAR no banco de dados local (sem consulta a APIs externas). Retorna a geometria do CAR e cruza automaticamente com parcelas SIGEF e certificados SNCI que intersectam a area.

<ParamField path="codigo_car" type="string" required>
  Codigo do CAR (ex: `SP-3545803-FD2A9A3E60834C0E9D74543B0184A4E7`).
</ParamField>

<ParamField query="min_sobreposicao" type="number" default={10.0}>
  Percentual minimo de sobreposicao para considerar um match (0 a 100).
</ParamField>

## Response

<ResponseField name="data" type="object">
  <Expandable title="data">
    <ResponseField name="car_data" type="object">
      Dados e geometria do CAR encontrado.
    </ResponseField>

    <ResponseField name="sigef_data" type="array">
      Lista de parcelas SIGEF que intersectam o CAR.
    </ResponseField>

    <ResponseField name="snci_data" type="array">
      Lista de certificados SNCI que intersectam o CAR.
    </ResponseField>

    <ResponseField name="principal_sigef" type="object">
      Parcela SIGEF com maior sobreposicao. `null` se nenhuma encontrada.
    </ResponseField>

    <ResponseField name="principal_snci" type="object">
      Certificado SNCI com maior sobreposicao. `null` se nenhum encontrado.
    </ResponseField>

    <ResponseField name="sobreposicao" type="object">
      Metricas de sobreposicao (area de intersecao, percentuais).
    </ResponseField>

    <ResponseField name="total_sigef" type="integer">
      Total de parcelas SIGEF encontradas.
    </ResponseField>

    <ResponseField name="total_snci" type="integer">
      Total de certificados SNCI encontrados.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://data-api.dadosfazenda.com.br/car/local/SP-3545803-FD2A9A3E60834C0E9D74543B0184A4E7" \
  -H "Authorization: Bearer SEU_TOKEN"
```

```python Python
import httpx

response = httpx.get(
    "https://data-api.dadosfazenda.com.br/car/local/SP-3545803-FD2A9A3E60834C0E9D74543B0184A4E7",
    headers={"Authorization": "Bearer SEU_TOKEN"}
)
data = response.json()["data"]
print(f"SIGEF encontrados: {data['total_sigef']}")
```
</RequestExample>
