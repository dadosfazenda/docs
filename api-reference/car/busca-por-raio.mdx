---
title: "Busca CAR por Raio"
api: "GET /car/search/radius"
description: "Busca CARs dentro de um raio a partir de uma coordenada geografica"
---

## Descricao

Busca todos os CARs cujas geometrias estejam dentro de um raio (em km) a partir de um ponto central definido por latitude e longitude.

<ParamField query="lat" type="number" required>
  Latitude do ponto central (-90 a 90).
</ParamField>

<ParamField query="lng" type="number" required>
  Longitude do ponto central (-180 a 180).
</ParamField>

<ParamField query="radius_km" type="number" default={10}>
  Raio de busca em quilometros (max: 100).
</ParamField>

<ParamField query="offset" type="integer" default={0}>
  Offset para paginacao.
</ParamField>

<ParamField query="limit" type="integer">
  Quantidade de registros por pagina. Se nao informado, retorna todos.
</ParamField>

## Response

<ResponseField name="data" type="object">
  <Expandable title="data">
    <ResponseField name="cars" type="array">
      Lista de CARs encontrados dentro do raio, cada um com codigo, municipio, estado, area e geometria.
    </ResponseField>

    <ResponseField name="metadata" type="object">
      <Expandable title="metadata">
        <ResponseField name="total" type="integer">Total de CARs encontrados.</ResponseField>
        <ResponseField name="offset" type="integer">Offset atual.</ResponseField>
        <ResponseField name="limit" type="integer">Limite por pagina.</ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="query" type="object">
      <Expandable title="query">
        <ResponseField name="lat" type="number">Latitude informada.</ResponseField>
        <ResponseField name="lng" type="number">Longitude informada.</ResponseField>
        <ResponseField name="radius_km" type="number">Raio utilizado.</ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://data-api.dadosfazenda.com.br/car/search/radius?lat=-15.79&lng=-47.88&radius_km=5" \
  -H "Authorization: Bearer SEU_TOKEN"
```

```python Python
import httpx

response = httpx.get(
    "https://data-api.dadosfazenda.com.br/car/search/radius",
    params={"lat": -15.79, "lng": -47.88, "radius_km": 5},
    headers={"Authorization": "Bearer SEU_TOKEN"}
)
cars = response.json()["data"]["cars"]
print(f"Encontrados {len(cars)} CARs no raio")
```
</RequestExample>
