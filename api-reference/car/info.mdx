---
title: "Informacoes do CAR"
api: "GET /car/{codigo_car}/info"
description: "Dados completos do CAR incluindo demonstrativo ambiental (reserva legal, APP, vegetacao nativa)"
---

## Descricao

Retorna dados completos de um CAR incluindo o demonstrativo ambiental do site car.gov.br.
O demonstrativo contem informacoes detalhadas sobre reserva legal, APP, vegetacao nativa, uso consolidado e sobreposicoes.

<ParamField path="codigo_car" type="string" required>
  Codigo do CAR. Exemplo: `GO-5203401-617CD0C0367A49CD809862F382945511`
</ParamField>

<ParamField query="min_sobreposicao" type="number" default={10.0}>
  Percentual minimo de sobreposicao com SIGEF (0 a 100).
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://data-api.dadosfazenda.com.br/car/GO-5203401-617CD0C0367A49CD809862F382945511/info" \
  -H "Authorization: Bearer SEU_TOKEN_API"
```

```python Python
import httpx

response = httpx.get(
    "https://data-api.dadosfazenda.com.br/car/GO-5203401-617CD0C0367A49CD809862F382945511/info",
    headers={"Authorization": "Bearer SEU_TOKEN"}
)
data = response.json()["data"]
print(f"CAR: {data['dados_car']['codigo']}")

demo = data.get("demonstrativo")
if demo:
    rl = demo["areas"]["reserva_legal"]
    print(f"Reserva Legal proposta: {rl['area_rl_proposta_ha']} ha")
    print(f"Reserva Legal a recompor: {rl['area_rl_recompor_ha']} ha")
    print(f"APP: {demo['areas']['app']['area_app_ha']} ha")
```
