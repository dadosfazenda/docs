---
title: "Serie Temporal NDVI"
api: "GET /ndvi/{property_type}/{property_code}/timeseries"
description: "Serie temporal de NDVI (indice de vegetacao) para uma propriedade rural"
---

## Descricao

Retorna a serie temporal do indice NDVI (Normalized Difference Vegetation Index) para uma propriedade rural. Primeiro verifica o cache; se nao houver dados ou `force=true`, recomputa a partir de imagens Sentinel-2.

O NDVI varia de -1 a 1, onde valores proximos a 1 indicam vegetacao saudavel e densa.

**Fonte**: Sentinel-2 (Planetary Computer)
**Resolucao**: 10 metros

<ParamField path="property_type" type="string" required>
  Tipo da propriedade: `car`, `sigef` ou `snci`.
</ParamField>

<ParamField path="property_code" type="string" required>
  Codigo da propriedade (CAR code, SIGEF UUID, ou SNCI certificate number).
</ParamField>

<ParamField query="days" type="integer" default={90}>
  Periodo historico em dias (min: 7, max: 365).
</ParamField>

<ParamField query="periods" type="integer" default={7}>
  Numero de periodos de analise (min: 2, max: 12).
</ParamField>

<ParamField query="force" type="boolean" default={false}>
  Forcar recomputacao a partir de satelite (ignora cache).
</ParamField>

<ParamField query="images" type="boolean" default={false}>
  Gerar imagens RGB e NDVI com URLs assinadas.
</ParamField>

## Response

<ResponseField name="data" type="object">
  <Expandable title="data">
    <ResponseField name="property_code" type="string">Codigo da propriedade.</ResponseField>
    <ResponseField name="property_type" type="string">Tipo resolvido (`car`, `sigef`, `snci`).</ResponseField>
    <ResponseField name="geometry_area_ha" type="number">Area total em hectares.</ResponseField>
    <ResponseField name="cached" type="boolean">Se veio do cache.</ResponseField>
    <ResponseField name="timeseries" type="array">
      Array de datapoints NDVI:
      <Expandable title="datapoint">
        <ResponseField name="scene_date" type="string">Data da observacao `YYYY-MM-DD`.</ResponseField>
        <ResponseField name="ndvi_mean" type="number">NDVI medio (-1 a 1).</ResponseField>
        <ResponseField name="ndvi_std" type="number">Desvio padrao do NDVI.</ResponseField>
        <ResponseField name="rgb_url" type="string">URL da imagem RGB (se `images=true`).</ResponseField>
        <ResponseField name="ndvi_url" type="string">URL da visualizacao NDVI (se `images=true`).</ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://data-api.dadosfazenda.com.br/ndvi/car/GO-5203401-617CD0C0/timeseries?days=180&periods=10" \
  -H "Authorization: Bearer SEU_TOKEN"
```

```python Python
import httpx

response = httpx.get(
    "https://data-api.dadosfazenda.com.br/ndvi/car/GO-5203401-617CD0C0/timeseries",
    params={"days": 180, "periods": 10},
    headers={"Authorization": "Bearer SEU_TOKEN"}
)
data = response.json()["data"]
for point in data["timeseries"]:
    print(f"{point['scene_date']}: NDVI = {point['ndvi_mean']:.3f}")
```
</RequestExample>
