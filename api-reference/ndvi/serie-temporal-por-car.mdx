---

## Descricao

Retorna a serie temporal do indice de vegetacao (NDVI) para uma propriedade rural a partir do codigo CAR.
O NDVI e calculado a partir de imagens do satelite Sentinel-2 com resolucao de 10 metros.

**Periodo maximo permitido: 90 dias.**

Quando `images=true`, inclui URLs assinadas para imagens RGB e NDVI processadas (validade de 1 hora).

Fonte: Sentinel-2 via Planetary Computer (Microsoft). Resolucao: 10m. Mascaramento de nuvens automatico.

<ParamField path="codigo_car" type="string" required>
  Codigo do CAR. Exemplo: `MT-5106505-EC68A6B4C2AF4EB283BCE84BF6D56A17`
</ParamField>

<ParamField query="start_date" type="string">
  Data inicio no formato `YYYY-MM-DD`. Default: 90 dias atras.
</ParamField>

<ParamField query="end_date" type="string">
  Data fim no formato `YYYY-MM-DD`. Default: hoje. Maximo de 90 dias de intervalo.
</ParamField>

<ParamField query="images" type="boolean" default={false}>
  Se `true`, gera imagens de satelite (RGB e NDVI) e retorna URLs assinadas com validade de 1 hora.
</ParamField>

<ParamField query="force" type="boolean" default={false}>
  Se `true`, recalcula o NDVI mesmo se houver dados em cache.
</ParamField>

## Response

  <Expandable title="data">
    <ResponseField name="property_code" type="string">
      Codigo do CAR consultado.
    </ResponseField>

    <ResponseField name="property_type" type="string">
      Tipo de propriedade (sempre `car` nesta rota).
    </ResponseField>

    <ResponseField name="start_date" type="string">
      Data inicio do periodo consultado.
    </ResponseField>

    <ResponseField name="end_date" type="string">
      Data fim do periodo consultado.
    </ResponseField>

    <ResponseField name="geometry_area_ha" type="number">

        </ResponseField>

        <ResponseField name="ndvi_mean" type="number">

        </ResponseField>

        <ResponseField name="ndvi_median" type="number">
          NDVI mediano.
        </ResponseField>

        <ResponseField name="ndvi_min" type="number">
          NDVI minimo.
        </ResponseField>

        <ResponseField name="ndvi_max" type="number">
          NDVI maximo.
        </ResponseField>

        <ResponseField name="ndvi_std" type="number">

        </ResponseField>

        </ResponseField>

    </ResponseField>
  </Expandable>

### Escala NDVI

| Faixa | Classificacao |
|-------|--------------|
| -1.0 a 0.0 | Agua, solo exposto, areas urbanas |
| 0.0 a 0.2 | Solo exposto, vegetacao muito escassa |
| 0.2 a 0.4 | Vegetacao rala ou estressada |
| 0.4 a 0.6 | Vegetacao moderada |
| 0.6 a 0.8 | Vegetacao densa e saudavel |
| 0.8 a 1.0 | Vegetacao muito densa |

<RequestExample>
```bash cURL
curl -X GET "https://data-api.dadosfazenda.com.br/ndvi/car/MT-5106505-EC68A6B4C2AF4EB283BCE84BF6D56A17/timeseries?start_date=2025-01-01&end_date=2025-03-01&images=true" \
  -H "Authorization: Bearer SEU_TOKEN_API"
```

```python Python
import httpx

response = httpx.get(
    "https://data-api.dadosfazenda.com.br/ndvi/car/MT-5106505-EC68A6B4C2AF4EB283BCE84BF6D56A17/timeseries",
    params={
        "start_date": "2025-01-01",
        "end_date": "2025-03-01",
        "images": True,
    },
    headers={"Authorization": "Bearer SEU_TOKEN"}
)
data = response.json()["data"]
for point in data["timeseries"]:
    print(f"{point['scene_date']}: NDVI={point['ndvi_mean']:.3f}")
```
