---
title: "Busca por CPF/CNPJ"
api: "GET /cpf-car/{cpf_cnpj}"
description: "Busca propriedades rurais (CAR) vinculadas a um CPF ou CNPJ"
---

## Descricao

A partir de um CPF ou CNPJ, esta rota identifica todos os imoveis rurais vinculados ao documento consultando bases publicas oficiais.
Para cada imovel encontrado, busca o CAR (Cadastro Ambiental Rural) correspondente com dados basicos do cadastro.

<Note>
  Para obter o demonstrativo ambiental detalhado de um CAR (reserva legal, APP, vegetacao nativa),
  utilize a rota [Obter informacoes de um CAR](/api-reference/car/info).
</Note>

<ParamField path="cpf_cnpj" type="string" required>
  CPF (11 digitos) ou CNPJ (14 digitos). Aceita com ou sem formatacao (pontos, tracos, barras).

  Exemplos: `12345678901`, `123.456.789-01`, `12345678000199`
</ParamField>

## Response

<ResponseField name="message" type="string">
  Mensagem descritiva do resultado.
</ResponseField>

<ResponseField name="data" type="object">
  <Expandable title="data">
    <ResponseField name="cpf_cnpj" type="string">
      Documento consultado (somente digitos).
    </ResponseField>

    <ResponseField name="imoveis_incra" type="integer">
      Total de imoveis rurais encontrados no INCRA para este documento.
    </ResponseField>

    <ResponseField name="cars_encontrados" type="integer">
      Total de CARs que foram vinculados com sucesso.
    </ResponseField>

    <ResponseField name="propriedades" type="array">
      Lista de propriedades encontradas.

      <Expandable title="propriedade">
        <ResponseField name="codigo_imovel_incra" type="string">
          Codigo do imovel no INCRA (ex: `433133781`).
        </ResponseField>

        <ResponseField name="denominacao" type="string">
          Nome do imovel (ex: `Fazenda Esperanca`).
        </ResponseField>

        <ResponseField name="municipio" type="string">
          Municipio sede do imovel segundo o INCRA.
        </ResponseField>

        <ResponseField name="uf" type="string">
          UF do imovel (ex: `GO`).
        </ResponseField>

        <ResponseField name="area_total_ha" type="number">
          Area total do imovel em hectares.
        </ResponseField>

        <ResponseField name="match_method" type="string">
          Metodo de vinculacao ao CAR: `sigef`, `matricula`, `sncr` ou `nenhum`.
        </ResponseField>

        <ResponseField name="ccir" type="object">
          Resumo dos dados do CCIR (Certificado de Cadastro de Imovel Rural).

          <Expandable title="ccir">
            <ResponseField name="denominacao" type="string">
              Nome do imovel.
            </ResponseField>

            <ResponseField name="classificacao_fundiaria" type="string">
              Classificacao (ex: `GRANDE PROPRIEDADE`).
            </ResponseField>

            <ResponseField name="area_total" type="number">
              Area total em hectares.
            </ResponseField>

            <ResponseField name="area_certificada" type="number">
              Area certificada pelo INCRA em hectares.
            </ResponseField>

            <ResponseField name="numero_modulos_fiscais" type="number">
              Quantidade de modulos fiscais.
            </ResponseField>

            <ResponseField name="titulares" type="array">
              Lista de titulares do imovel.

              <Expandable title="titular">
                <ResponseField name="nome" type="string">
                  Nome do titular.
                </ResponseField>
                <ResponseField name="cpf_cnpj" type="string">
                  CPF ou CNPJ do titular.
                </ResponseField>
                <ResponseField name="condicao" type="string">
                  Condicao de titularidade (ex: `PROPRIETARIO`).
                </ResponseField>
                <ResponseField name="percentual" type="number">
                  Percentual de detencao do imovel.
                </ResponseField>
              </Expandable>
            </ResponseField>

            <ResponseField name="ccir" type="object">
              Dados do ultimo CCIR emitido.

              <Expandable title="ccir_dados">
                <ResponseField name="numero" type="string">
                  Numero do CCIR.
                </ResponseField>
                <ResponseField name="situacao" type="string">
                  Situacao do CCIR (ex: `REGULAR`).
                </ResponseField>
                <ResponseField name="data_vencimento" type="string">
                  Data de vencimento (formato `YYYY-MM-DD`).
                </ResponseField>
              </Expandable>
            </ResponseField>
          </Expandable>
        </ResponseField>

        <ResponseField name="car" type="object">
          Dados do CAR vinculado. `null` se nenhum CAR foi encontrado.

          <Expandable title="car">
            <ResponseField name="codigo" type="string">
              Codigo do CAR (ex: `GO-5203401-617CD0C0367A49CD809862F382945511`).
            </ResponseField>

            <ResponseField name="municipio" type="string">
              Municipio do CAR.
            </ResponseField>

            <ResponseField name="estado" type="string">
              UF do CAR (ex: `GO`).
            </ResponseField>

            <ResponseField name="area_ha" type="number">
              Area do CAR em hectares.
            </ResponseField>

            <ResponseField name="modulos_fiscais" type="number">
              Quantidade de modulos fiscais.
            </ResponseField>

            <ResponseField name="status" type="string">
              Status do CAR: `AT` (Ativo), `PE` (Pendente), `CA` (Cancelado), `SU` (Suspenso).
            </ResponseField>

            <ResponseField name="tipo" type="string">
              Tipo do imovel: `IRU` (Imovel Rural), `AST` (Assentamento), `PCT` (Territorio Tradicional).
            </ResponseField>

            <ResponseField name="condicao" type="string">
              Condicao de analise (ex: `Aguardando analise`, `Analisado`).
            </ResponseField>
          </Expandable>
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://data-api.dadosfazenda.com.br/cpf-car/12345678901" \
  -H "Authorization: Bearer SEU_TOKEN_API"
```

```python Python
import httpx

response = httpx.get(
    "https://data-api.dadosfazenda.com.br/cpf-car/12345678901",
    headers={"Authorization": "Bearer SEU_TOKEN"}
)
data = response.json()

for prop in data["data"]["propriedades"]:
    print(f"{prop['denominacao']} - {prop['municipio']}/{prop['uf']}")
    if prop["car"]:
        print(f"  CAR: {prop['car']['codigo']}")
        print(f"  Area: {prop['car']['area_ha']} ha")
```
</RequestExample>
