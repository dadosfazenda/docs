---
title: "Busca por CPF/CNPJ"
api: "GET /cpf-car/{cpf_cnpj}"
description: "Busca propriedades rurais (CAR) vinculadas a um CPF ou CNPJ"
---

## Descricao

A partir de um CPF ou CNPJ, esta rota identifica todos os imoveis rurais vinculados ao documento consultando bases publicas oficiais.
Para cada imovel encontrado, busca o CAR (Cadastro Ambiental Rural) correspondente e enriquece com dados do demonstrativo ambiental,
incluindo informacoes sobre reserva legal, APP, vegetacao nativa e sobreposicoes.

<ParamField path="cpf_cnpj" type="string" required>
  CPF (11 digitos) ou CNPJ (14 digitos). Aceita com ou sem formatacao (pontos, tracos, barras).

  Exemplos: `12345678901`, `123.456.789-01`, `12345678000199`
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://data-api.dadosfazenda.com.br/cpf-car/12345678901" \
  -H "Authorization: Bearer SEU_TOKEN"
```

```python Python
import httpx

response = httpx.get(
    "https://data-api.dadosfazenda.com.br/cpf-car/12345678901",
    headers={"Authorization": "Bearer SEU_TOKEN"}
)
data = response.json()

for prop in data["data"]["propriedades"]:
    print(f"{prop['denominacao']} - {prop['municipio']}/{prop['uf']}")
    if prop["car"]:
        print(f"  CAR: {prop['car']['codigo']}")
        demo = prop["car"].get("demonstrativo")
        if demo:
            rl = demo["areas"]["reserva_legal"]
            print(f"  RL proposta: {rl['area_rl_proposta_ha']} ha")
            print(f"  RL a recompor: {rl['area_rl_recompor_ha']} ha")
```

```javascript JavaScript
const response = await fetch(
  "https://data-api.dadosfazenda.com.br/cpf-car/12345678901",
  { headers: { Authorization: "Bearer SEU_TOKEN" } }
);
const { data } = await response.json();

data.propriedades.forEach((prop) => {
  console.log(`${prop.denominacao} - ${prop.municipio}/${prop.uf}`);
  if (prop.car?.demonstrativo) {
    const rl = prop.car.demonstrativo.areas.reserva_legal;
    console.log(`  RL proposta: ${rl.area_rl_proposta_ha} ha`);
  }
});
```
