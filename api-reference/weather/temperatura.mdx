---
title: "Temperatura"
api: "GET /weather/temperature"
description: "Temperatura diaria (maxima e minima) para uma coordenada e periodo"
---

## Descricao

Retorna temperatura maxima e minima diaria em graus Celsius.

**Fonte**: Open-Meteo Historical Weather API (ERA5)
**Resolucao**: ~25 km | **Dados**: 1940 ate ~5 dias atras

<ParamField query="latitude" type="number" required>Latitude (-90 a 90).</ParamField>
<ParamField query="longitude" type="number" required>Longitude (-180 a 180).</ParamField>
<ParamField query="start_date" type="string" required>Data inicio `YYYY-MM-DD`.</ParamField>
<ParamField query="end_date" type="string" required>Data fim `YYYY-MM-DD`.</ParamField>

## Response

<ResponseField name="data" type="object">
  <Expandable title="data">
    <ResponseField name="dates" type="array">Datas `YYYY-MM-DD`.</ResponseField>
    <ResponseField name="temperature_max" type="array">Temperatura maxima diaria em Celsius.</ResponseField>
    <ResponseField name="temperature_min" type="array">Temperatura minima diaria em Celsius.</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://data-api.dadosfazenda.com.br/weather/temperature?latitude=-15.79&longitude=-47.88&start_date=2024-01-01&end_date=2024-01-31" \
  -H "Authorization: Bearer SEU_TOKEN"
```
</RequestExample>
