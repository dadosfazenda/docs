---
title: "Precipitacao"
api: "GET /weather/precipitation"
description: "Precipitacao diaria acumulada (mm) para uma coordenada e periodo"
---

## Descricao

Retorna dados de precipitacao diaria acumulada em milimetros para um ponto geografico e periodo especificado.

**Fonte**: Open-Meteo Historical Weather API (ERA5 reanalysis)
**Resolucao espacial**: ~25 km
**Dados disponiveis**: 1940 ate ~5 dias atras

<ParamField query="latitude" type="number" required>
  Latitude (-90 a 90).
</ParamField>

<ParamField query="longitude" type="number" required>
  Longitude (-180 a 180).
</ParamField>

<ParamField query="start_date" type="string" required>
  Data inicio no formato `YYYY-MM-DD`.
</ParamField>

<ParamField query="end_date" type="string" required>
  Data fim no formato `YYYY-MM-DD`.
</ParamField>

## Response

<ResponseField name="data" type="object">
  <Expandable title="data">
    <ResponseField name="dates" type="array">
      Array de datas no formato `YYYY-MM-DD`.
    </ResponseField>
    <ResponseField name="precipitation" type="array">
      Array de precipitacao diaria em mm (alinhado por indice com `dates`).
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://data-api.dadosfazenda.com.br/weather/precipitation?latitude=-15.79&longitude=-47.88&start_date=2024-01-01&end_date=2024-01-31" \
  -H "Authorization: Bearer SEU_TOKEN"
```

```python Python
import httpx

response = httpx.get(
    "https://data-api.dadosfazenda.com.br/weather/precipitation",
    params={
        "latitude": -15.79,
        "longitude": -47.88,
        "start_date": "2024-01-01",
        "end_date": "2024-01-31"
    },
    headers={"Authorization": "Bearer SEU_TOKEN"}
)
data = response.json()["data"]
for date, mm in zip(data["dates"], data["precipitation"]):
    print(f"{date}: {mm} mm")
```
</RequestExample>
