---
title: "Temperatura por CAR"
api: "GET /weather/temperature/car/{codigo_car}"
description: "Historico de temperatura diaria (maxima e minima) para um CAR"
---

## Descricao

Retorna o historico de temperatura diaria (maxima e minima em graus Celsius) para o centroide de uma propriedade rural
a partir do codigo CAR.

**Periodo maximo permitido: 12 meses (366 dias).**

Fonte: Open-Meteo Historical Weather API (ERA5 reanalysis). Resolucao espacial: ~25 km.
Dados disponiveis de 1940 ate ~5 dias atras.

<ParamField path="codigo_car" type="string" required>
  Codigo do CAR. Exemplo: `GO-5205513-DAF91DDC4B054D37919111EA3E7322D3`
</ParamField>

<ParamField query="start_date" type="string" required>
  Data inicio no formato `YYYY-MM-DD`.
</ParamField>

<ParamField query="end_date" type="string" required>
  Data fim no formato `YYYY-MM-DD`. O intervalo entre `start_date` e `end_date` nao pode exceder 12 meses.
</ParamField>

## Response

<ResponseField name="data" type="object">
  <Expandable title="data">
    <ResponseField name="dates" type="array">
      Array de datas no formato `YYYY-MM-DD`.
    </ResponseField>

    <ResponseField name="temperature_max" type="array">
      Array de temperaturas maximas diarias em graus Celsius.
    </ResponseField>

    <ResponseField name="temperature_min" type="array">
      Array de temperaturas minimas diarias em graus Celsius.
    </ResponseField>

    <ResponseField name="car" type="object">
      Dados do CAR consultado.

      <Expandable title="car">
        <ResponseField name="codigo" type="string">Codigo do CAR.</ResponseField>
        <ResponseField name="municipio" type="string">Municipio.</ResponseField>
        <ResponseField name="estado" type="string">UF.</ResponseField>
        <ResponseField name="area_ha" type="number">Area em hectares.</ResponseField>
        <ResponseField name="centroid" type="object">
          <Expandable title="centroid">
            <ResponseField name="latitude" type="number">Latitude do centroide.</ResponseField>
            <ResponseField name="longitude" type="number">Longitude do centroide.</ResponseField>
          </Expandable>
        </ResponseField>
      </Expandable>
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://data-api.dadosfazenda.com.br/weather/temperature/car/GO-5205513-DAF91DDC4B054D37919111EA3E7322D3?start_date=2025-01-01&end_date=2025-03-01" \
  -H "Authorization: Bearer SEU_TOKEN_API"
```

```python Python
import httpx

response = httpx.get(
    "https://data-api.dadosfazenda.com.br/weather/temperature/car/GO-5205513-DAF91DDC4B054D37919111EA3E7322D3",
    params={"start_date": "2025-01-01", "end_date": "2025-03-01"},
    headers={"Authorization": "Bearer SEU_TOKEN"}
)
data = response.json()["data"]
for i, dt in enumerate(data["dates"]):
    print(f"{dt}: {data['temperature_min'][i]}C - {data['temperature_max'][i]}C")
```
</RequestExample>
