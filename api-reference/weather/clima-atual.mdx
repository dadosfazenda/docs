---
title: "Clima Atual"
api: "GET /weather/current"
description: "Temperatura, vento e umidade em tempo real para uma coordenada"
---

## Descricao

Retorna condicoes climaticas atuais para um ponto geografico.

**Fonte**: Open-Meteo Forecast API (dados em tempo real)

<ParamField query="latitude" type="number" required>Latitude (-90 a 90).</ParamField>
<ParamField query="longitude" type="number" required>Longitude (-180 a 180).</ParamField>

## Response

<ResponseField name="data" type="object">
  <Expandable title="data">
    <ResponseField name="temperature" type="number">Temperatura atual em Celsius.</ResponseField>
    <ResponseField name="weather_code" type="integer">Codigo WMO da condicao climatica.</ResponseField>
    <ResponseField name="wind_speed" type="number">Velocidade do vento em km/h.</ResponseField>
    <ResponseField name="wind_direction" type="integer">Direcao do vento em graus.</ResponseField>
    <ResponseField name="relative_humidity" type="integer">Umidade relativa em %.</ResponseField>
    <ResponseField name="timestamp" type="string">Data/hora da observacao (ISO 8601).</ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://data-api.dadosfazenda.com.br/weather/current?latitude=-15.79&longitude=-47.88" \
  -H "Authorization: Bearer SEU_TOKEN"
```
</RequestExample>
