---
title: "Delineamento de Talhoes"
api: "GET /delineate/{codigo_car}/fields"
description: "Retorna talhoes preditos por IA para um CAR usando deteccao por satelite"
---

## Descricao

Retorna os poligonos dos talhoes (fields) detectados por inteligencia artificial dentro da geometria de um CAR. Utiliza a API OneSoil para deteccao automatica a partir de imagens de satelite.

**Fonte**: OneSoil AI
**Base**: Imagens Sentinel-2

<ParamField path="codigo_car" type="string" required>
  Codigo do CAR.
</ParamField>

<ParamField query="limit" type="integer" default={100}>
  Limite de talhoes retornados (max: 500).
</ParamField>

## Response

<ResponseField name="data" type="object">
  <Expandable title="data">
    <ResponseField name="codigo_car" type="string">Codigo do CAR.</ResponseField>
    <ResponseField name="fields" type="array">
      Array de talhoes detectados, cada um com:
      - `geometry` (GeoJSON Polygon) — poligono do talhao
      - `area_ha` (number) — area em hectares
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://data-api.dadosfazenda.com.br/delineate/GO-5203401-617CD0C0/fields" \
  -H "Authorization: Bearer SEU_TOKEN"
```
</RequestExample>
