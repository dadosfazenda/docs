title: "Alertas Ambientais por CAR"
api: "GET /ambiental/car/{codigo_car}"
description: "Alertas de desmatamento, embargos IBAMA, reserva legal e APP via MapBiomas"
---

## Descricao

Retorna dados ambientais completos de um imovel rural consultando a API MapBiomas Alerta v2. Inclui alertas de desmatamento, embargos, sobreposicoes com areas protegidas e imagens de satelite antes/depois.

As imagens de satelite dos alertas sao cacheadas em nosso servidor e entregues via URLs assinadas com validade de **1 hora**.

**Fonte**: MapBiomas Alerta (plataforma.alerta.mapbiomas.org)

<ParamField path="codigo_car" type="string" required>
  Codigo do CAR. Exemplo: `MT-5106505-EC68A6B4C2AF4EB283BCE84BF6D56A17`
</ParamField>

<RequestExample>
```bash cURL
curl -X GET "https://data-api.dadosfazenda.com.br/ambiental/car/MT-5106505-EC68A6B4C2AF4EB283BCE84BF6D56A17" \
  -H "Authorization: Bearer SEU_TOKEN_API"
```

```python Python
import httpx

response = httpx.get(
    "https://data-api.dadosfazenda.com.br/ambiental/car/MT-5106505-EC68A6B4C2AF4EB283BCE84BF6D56A17",
    headers={"Authorization": "Bearer SEU_TOKEN"}
)
data = response.json()["data"]
resumo = data["resumo_ambiental"]
print(f"Alertas: {resumo['total_alertas_desmatamento']}")
print(f"Area desmatada: {resumo['area_total_desmatada_ha']} ha")
print(f"Possui embargo: {resumo['possui_embargo']}")

for alerta in data["alertas_desmatamento"]:
    print(f"  Alerta {alerta['codigo_alerta']}: {alerta['area_ha']} ha ({alerta['detectado_em']})")
    for img in alerta.get("imagens_satelite", []):
        print(f"    Imagem: {img['url']}")
```
</RequestExample>
