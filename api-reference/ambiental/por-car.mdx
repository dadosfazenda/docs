---
title: "Dados Ambientais por CAR"
api: "GET /ambiental/car/{codigo_car}"
description: "Alertas de desmatamento, embargos IBAMA, reserva legal e APP via MapBiomas"
---

## Descricao

Retorna dados ambientais completos de um imovel rural consultando a API MapBiomas Alerta v2. Inclui alertas de desmatamento, embargos, sobreposicoes com areas protegidas e imagens de satelite antes/depois.

**Fonte**: MapBiomas Alerta (plataforma.alerta.mapbiomas.org)

<Note>
  Este endpoint e publico e **nao requer autenticacao**.
</Note>

<ParamField path="codigo_car" type="string" required>
  Codigo do CAR.
</ParamField>

## Response

  <Expandable title="data">
    <ResponseField name="imovel" type="object">

        <ResponseField name="versao_car" type="string">Versao do CAR.</ResponseField>

    </ResponseField>

        <ResponseField name="total_sobreposicoes" type="integer">Total de sobreposicoes com areas protegidas.</ResponseField>

        <ResponseField name="velocidade" type="string">Categoria de velocidade do desmatamento.</ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="reserva_legal" type="object">
      `area_ha`, `geometrias` (GeoJSON).
    </ResponseField>

    <ResponseField name="app" type="object">
      `area_ha`, `geometrias` (GeoJSON).
    </ResponseField>

    <ResponseField name="embargos" type="object">
      `area_ha`, `detalhes` (array de embargos IBAMA).
    </ResponseField>

    <ResponseField name="autorizacoes_desmatamento" type="array">
      Autorizacoes de supressao vegetal.
    </ResponseField>

    <ResponseField name="territorios_cruzados" type="object">

        <ResponseField name="biomas" type="array">Biomas sobrepostos.</ResponseField>

        <ResponseField name="estados" type="array">Estados.</ResponseField>
        <ResponseField name="unidades_conservacao" type="array">Unidades de conservacao.</ResponseField>
        <ResponseField name="terras_indigenas" type="array">Terras indigenas.</ResponseField>
        <ResponseField name="quilombos" type="array">Comunidades quilombolas.</ResponseField>
        <ResponseField name="assentamentos" type="array">Assentamentos.</ResponseField>
      </Expandable>
    </ResponseField>

    <ResponseField name="imagens_satelite" type="array">
      Imagens de satelite antes/depois dos alertas.
    </ResponseField>

    <ResponseField name="nascentes" type="array">
      Nascentes d'agua dentro da propriedade.
    </ResponseField>
  </Expandable>
</ResponseField>

<RequestExample>
```bash cURL
curl -X GET "https://data-api.dadosfazenda.com.br/ambiental/car/GO-5203401-617CD0C0"
```

```python Python
import httpx

response = httpx.get(
    "https://data-api.dadosfazenda.com.br/ambiental/car/GO-5203401-617CD0C0"
)
data = response.json()["data"]
resumo = data["resumo_ambiental"]
print(f"Alertas: {resumo['total_alertas']}")
print(f"Area desmatada: {resumo['area_desmatada_ha']} ha")
print(f"Area embargada: {resumo['area_embargada_ha']} ha")
```
</RequestExample>
