# DadosFazenda Data API

A DadosFazenda Data API permite consultar dados de propriedades rurais brasileiras de forma unificada, cruzando informacoes de diversas fontes oficiais.

## Fontes de dados

| Fonte | Dados |
|-------|-------|
| **SICAR / CAR** | Cadastro Ambiental Rural (geometria, status, demonstrativo ambiental) |
| **Serpro / CCIR** | Imoveis rurais vinculados ao CPF/CNPJ, titulares, classificacao fundiaria |
| **MapBiomas Alerta** | Desmatamento, embargos, areas protegidas, imagens de satelite |
| **Open-Meteo** | Precipitacao e temperatura historica (ERA5 reanalysis) |
| **SoilGrids (ISRIC)** | Propriedades do solo em resolucao de 250m |
| **Sentinel-2** | Imagens de satelite e NDVI em resolucao de 10m |

## Formato de resposta

```json Sucesso
{
  "success": true,
  "message": "Descricao do resultado",
  "data": { ... }
}
```

```json Erro
{
  "success": false,
  "message": "Descricao do erro",
  "data": null
}
```

| Codigo | Significado |
|--------|-------------|
| `400` | Parametros invalidos |
| `401` | Token ausente ou invalido |
| `403` | API key desativada |
| `404` | Recurso nao encontrado |
| `429` | Limite mensal de requests atingido |
| `502` | Erro em API externa |
